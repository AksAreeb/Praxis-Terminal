# Praxis Terminal - Alpha Execution Framework

**Praxis Terminal** is a low latency execution wrapper and risk management protocol designed for the Interactive Brokers (IBKR) API. It provides a secure bridge for routing algorithmic trades into complex derivatives and event driven markets, mathematically shielding the account from catastrophic drawdown.

## Core Infrastructure

This repository contains the physical routing and capital preservation layers of a systematic trading operation. It is built to safely execute signals generated by external, high frequency intelligence models.

*   **API Gateway Abstraction:** Wraps the complex IBKR Java Client Portal into a clean, resilient Node.js environment. Handles authentication and seamless endpoint routing.
*   **Deterministic Guardrails:** Hardcoded math that physically blocks the terminal from violating risk limits (e.g., maximum position sizes, rapid fire trade cooldowns).
*   **Dynamic Capital Constraints:** Calculates daily spending power as a live percentage of the account's total `NetLiquidation`. This feature enables external AI or statistical models to scale their bet sizes geometrically (using formulas like the Kelly Criterion) without manual intervention.

> **Proprietary Notice:** This repository contains *only* the structural execution bridge and the mathematical safety nets. The core quantitative strategy, including NLP sentiment engines, multi source alpha generators, statistical edge calculators and macroeconomic causality maps are maintained in isolated, off-network repositories and strictly ignored via `.gitignore` to protect the trading edge.

## Local Installation

1.  **Clone the infrastructure core.**
2.  **Install Gateway Dependencies:** (Requires Node v20+)
    ```bash
    npm install
    ```
3.  **Authentication:** Configure your localized `.env` variables and initialize the Java Vert.x Gateway locally on `localhost:5000`.
4.  **Engage Routing:**
    ```bash
    npx tsx src/index.ts
    ```

## Risk Compliance
Praxis Terminal operates under the strict assumption that external predictive models will eventually produce false signals. The execution engine enforces capital preservation as its highest directive. It will automatically sever API routing and block trades if predefined risk or ruin thresholds are breached.
